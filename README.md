# Superheroes API

## Description
A Flask API for managing heroes and their powers.

## Setup
- pipenv shell
- pipenv install
- flask db init
- flask db migrate -m "initial migration"
- flask db upgrade
- python seed.py
- flask run


## Routes
- GET /heroes
- GET /heroes/:id
- GET /powers
- GET /powers/:id
- PATCH /powers/:id
- POST /hero_powers
